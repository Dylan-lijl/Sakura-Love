# 🌸 Sakura Love

> 樱花飘落中的浪漫告白互动页面

一个基于 HTML / CSS / JavaScript 实现的情人节互动网页。

页面融合了：

* 🌸 樱花飘落动画
* ❤️ 爱心 SVG 路径文字动画
* 💌 情人节祝福语
* 🌷 鲜花背景弹窗
* ✨ 动态入场 / 退场效果
* 🎮 花瓣点击积分互动
* 🏃 会逃跑的“拒绝”按钮

适用于：

* 情人节页面
* 表白网页
* 前端动画练习
* 创意互动 H5
* 浪漫前端作品集

---

# 🚀 用法

直接打开：

```txt
index.html
```

即可运行项目。

无需安装任何依赖。
